Leave-one-out PI control (results doc S14 centerpiece)
======================================================================
Claim: exact-PI train_pos signal is a train-graph-membership
artifact. For each sampled train_pos edge (u,v) we compute its
vicinity PI WITH the (u,v) edge in the graph (train/cache
condition) vs WITHOUT it (held-out / test-like condition).
Identical PI math (sg2dgm hop=1 intersection, descriptor=sum,
resolution=5, weighted+normed); exact per-vicinity Ollivier-
Ricci curvature both conditions. Only the edge presence differs.
